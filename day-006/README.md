### While loop

- `For` do something to each item, or in a range.
- `While` do something repeatedly if something is true, and it's only when **something becomes false does the loop stop**.

#### Examples:

```python
while at_goal != True:
  # jump()
```

```python
while not at_goal:
  # jump()
```

- Checking while loop

```python
while 5 > 3:
  print(5 > 3) # True
  jump()
```
